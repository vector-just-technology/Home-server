Features

Everything your home needs

From AI chat to disk management — a complete operating system for your server.

🤖

AI Studio

Chat with local or cloud AI models. Generate code, analyze files, create images — all in one interface with conversation history.

💾

Storage Manager

Monitor drives, SMART data, ZFS pools, and RAID status. Safe auto-detection keeps your OS drive protected from accidental operations.

📱

Device Hub

Auto-discover devices via ARP + nmap. Organize by room, ping check, and 29 device type icons with color coding.

📊

Live Dashboard

Real-time CPU, RAM, and disk charts. Top processes, alerts, network monitoring, animated counters with sparklines.

🔧

System Tools

100+ tools across 20 categories: services, Docker, cron, firewall, USB, kernel modules, packages, sensors, and more.

👥

User Management

Granular per-user permissions — 14 pages × 40+ actions. Feature toggles, storage limits, and a restrictiveness meter.

🎨

Customization

24 accent colors, 54 wallpapers, custom image upload. Typography, layout density, and custom CSS for the entire UI.

🔒

Security

Encryption, automated backup jobs, audit logs, share links with password protection, and role-based access control.

🌐

WiFi Hotspot

Built-in access point mode. Share your connection or create a standalone network with the integrated hotspot manager.

Get Started

Install in one command

Everything is automated — Python, Node.js, database, and systemd service. Just run and go.

1

Run the installer

One command installs the entire server on your Raspberry Pi.

curl -sL https://raw.githubusercontent.com/vector-just-technology/Home-server/Stable/install.sh | bash
2

Open the dashboard

Access your server from any browser on your network.

open http://<pi-ip>:5000
3

Create your account

First user becomes the administrator. Invite others with granular permissions.

Click "Create one" → Done!
